# 문제 정의 및 프로젝트 개요

## 서비스 선정 배경

연애 고민은 많은 사용자가 쉽게 공감할 수 있는 주제이며, 정답이 하나로 정해지기 어렵다. 같은 상황이라도 사람마다 해석과 조언이 달라질 수 있기 때문에 멀티 Agent 구조를 보여주기에 적합하다.

기존의 일반 챗봇 상담은 하나의 관점으로 답변하는 경우가 많다. 반면 본 프로젝트는 여러 Agent가 서로 다른 연애관을 바탕으로 의견을 제시하고, 서로의 의견을 검토하며 최종 결론을 도출하는 구조를 가진다.

사용자는 단순히 결과 답변만 보는 것이 아니라, 각 Agent가 어떤 관점으로 상황을 해석했는지와 의견이 어떻게 충돌하고 정리되는지를 함께 확인할 수 있다.

## 해결하려는 문제

| 문제 | 설명 |
| --- | --- |
| 단일 관점 상담의 한계 | 하나의 답변만 제공될 경우 사용자가 다양한 해석을 비교하기 어렵다. |
| 연애 고민의 주관성 | 연애 문제는 사람마다 판단 기준이 다르기 때문에 하나의 정답으로 설명하기 어렵다. |
| 고민 정리의 어려움 | 사용자가 감정적으로 혼란스러운 상태에서는 상황을 객관적으로 정리하기 어렵다. |
| 기존 챗봇의 단조로움 | 일반 상담형 챗봇은 답변 과정이 보이지 않아 재미 요소와 몰입감이 부족하다. |

## 프로젝트 방향

본 프로젝트는 연애 고민이라는 주관적인 문제를 여러 Agent가 각자의 관점으로 분석하고 토론하게 함으로써, 멀티 Agent 구조의 장점인 다양한 관점, 상호 검토, 최종 합의 과정을 보여주는 것을 목표로 한다.
