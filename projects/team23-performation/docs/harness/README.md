# Harness Docs

This directory holds durable harness specs for Performation.

- `performation/team-spec.md`: role topology, phase order, handoffs, and failure policy
- `performation/output-contract.md`: guide JSON shape, user-facing sections, and confidence labels
- `performation/scenario-matrix.md`: normal and failure scenarios for demo QA
- `performation/git-policy.md`: commit splitting, message types, and pre-commit checks

Keep root `AGENTS.md` short and put conditional workflow details here.
